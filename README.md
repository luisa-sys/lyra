# Lyra - checklyra.com

A calm, structured public profile platform where people share preferences, gift ideas, boundaries, and personal details.

## Environments
- **Production**: https://checklyra.com
- **Staging**: https://stage.checklyra.com
- **Development**: https://dev.checklyra.com

## Tech Stack
- Next.js 16 + TypeScript + Tailwind CSS
- Supabase (PostgreSQL + Auth)
- Vercel (Hosting)
- Cloudflare (DNS)
